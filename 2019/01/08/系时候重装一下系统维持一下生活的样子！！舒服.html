工具准备

  1. 系统：windows10
  2. 环境和ide：jdk，eclipse，maven，mysql，HBuilerX，git for windows，sublime，node.js，powerdesigner，Android Studio，Android SDK，ArcGis，AutoCAD
  3. 压缩：BandZip
  4. 视频音频播放器：Potplayer
  5. 音乐：网易云音乐
  6. 下载：迅雷x（下种子和链接），IDM下载视频音频其他等等
  7. 聊天：微信、QQ（TIM）
  8. 浏览器：Chrome（扩展：Adblock Plus，Google 翻译，应用：postman）
  9. 科学上网：shadowsocksr，新界SSR
  10. office工具：微软的，需要破解
  11. 远程控制和文件传输：TeamViewer
  12. 云盘：百度网盘
  13. 娱乐：WeGame，DNF，哔哩哔哩客户端
  14. 驱动：驱动人生

开始干活

百度输入windows10，点击windows10下载那个网页，然后下爆，然后更新这台电脑，完事

个性化设置，BandZip压缩软件先装爆，IDM解压，科学上网解压，开爆科学上网去摸Chrome，应用商店摸爆

驱动人生下载非显卡驱动，然后去英伟达官网下载显卡驱动

Honeyview，Potplayer，网易云音乐，迅雷X，微信和TIM，百度云盘，哔哩哔哩客户端，TeamViewer，WeGame

环境和ide装爆，先jdk和mysql再其他

DNF下载，office下载，U盘文件，百度云文件，下爆

后台关爆，自启动关爆，win10无用东西卸载卸爆

具体环境配置

  1. jdk：下载安装，C盘默认x86文件，新建JAVA_HOME，值为jdk地址，然后在path编辑，加一行%JAVA_HOME%\bin，最后cmd输入java，javac，java-version看看情况
  2. mysql：密码root
  3. maven：新建MAVEN_HOME，值为maven地址，然后path编辑，加一行%MAVEN_HOME%\bin，在目录下用shift加右键打开shell然后mkdir .m2repository创建包路径，在目录下/conf/setting.xml修改如下：
<!-- localRepository
   | The path to the local repository maven will use to store artifacts.
   |
   | Default: ${user.home}/.m2/repository
  <localRepository>/path/to/local/repo</localRepository>
  -->

<!-- mirrors
 | This is a list of mirrors to be used in downloading artifacts from remote repositories.
 |
 | It works like this: a POM may declare a repository to use in resolving certain artifacts.
 | However, this repository may have problems with heavy traffic at times, so people have mirrored
 | it to several places.
 |
 | That repository definition will have a unique id, so we can create a mirror reference for that
 | repository, to be used as an alternate download site. The mirror site will be the preferred
 | server for that repository.
 |-->
<mirrors>
  <!-- mirror
   | Specifies a repository mirror site to use instead of a given repository. The repository that
   | this mirror serves has an ID that matches the mirrorOf element of this mirror. IDs are used
   | for inheritance and direct lookup purposes, and must be unique across the set of mirrors.
   |
  <mirror>
    <id>mirrorId</id>
    <mirrorOf>repositoryId</mirrorOf>
    <name>Human Readable Name for this Mirror.</name>
    <url>http://my.repository.com/repo/path</url>
  </mirror>
   -->
   <mirror>
    <id>alimaven</id>
    <name>aliyun maven</name>
    <url>http://maven.aliyun.com/nexus/content/groups/public/</url>
    <mirrorOf>central</mirrorOf>
  </mirror>
</mirrors>
  1. eclipse：配置maven（包括包路径），jdk，编码格式
  2. Android Studio：，AndroidSDK位置
  3. ArcGis：先装desktop再装server
  4. AutoCAD：破解破爆
  5. powerdesigner：破解破爆
  6. node.js：自动配置，去path看就完事了
  7. HBuilerX，git for windows，sublime：安装就完事了
